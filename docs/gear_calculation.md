# 档位计算说明

## 核心原理

通过转速和车速反推当前档位。

```
实时传动比 = (转速 × 60 × 轮胎周长) / (速度 × 1000)
```

传动比表示：发动机转多少圈，轮胎转一圈。

---

## 可调参数说明

### 1. 车辆参数

#### primaryRatio (初级减速比)
- 当前值: `1.681`
- 说明: 发动机输出轴到变速箱输入轴的减速比
- 来源: 车辆厂商规格，由发动机设计决定

#### finalRatio (最终减速比)
- 当前值: `2.764` (47/17)
- 说明: 变速箱输出轴到后轮的减速比
- 计算: 牙盘齿数 ÷ 链轮齿数 = 47 ÷ 17
- **如果改了牙盘，需要调整这个值**

#### tireCircumference (轮胎周长)
- 当前值: `1.97857` 米
- 对应规格: 180/55ZR17
- 说明: 后轮胎一圈的长度
- **如果改了轮胎尺寸，需要调整这个值**

#### internalGearRatios (各档内部齿轮比)
```dart
static const List<double> internalGearRatios = [
  3.076,  // 1档
  2.125,  // 2档
  1.619,  // 3档
  1.285,  // 4档
  1.083,  // 5档
  0.956,  // 6档
];
```
- 说明: 变速箱内部各档位的传动比
- 数值越大 = 传动越慢 = 扭矩越大 = 速度越慢
- **这是最可能需要校准的参数**

---

### 2. 各档理论总传动比

**总传动比 = primaryRatio × 该档齿轮比 × finalRatio**

| 档位 | 内部齿轮比 | 总传动比 |
|------|------------|----------|
| 1档 | 3.076 | 14.27 |
| 2档 | 2.125 | 9.86 |
| 3档 | 1.619 | 7.51 |
| 4档 | 1.285 | 5.96 |
| 5档 | 1.083 | 5.03 |
| 6档 | 0.956 | 4.44 |

**问题**: 4/5/6档传动比很接近（5.96 → 5.03 → 4.44），容易互相误判

---

### 3. 误差阈值 (_gearThresholds)

```dart
static const List<double> _gearThresholds = [
  0.20,  // 1档: 允许20%误差
  0.18,  // 2档: 允许18%误差
  0.15,  // 3档: 允许15%误差
  0.15,  // 4档: 允许15%误差
  0.15,  // 5档: 允许15%误差
  0.15,  // 6档: 允许15%误差
];
```

**作用**: 判断计算出的档位是否可信

例如：15% 阈值意味着
- 计算传动比 5.0，与理论 5.03 相差 0.03
- 相对误差 = 0.03 / 5.03 = 0.6%
- 0.6% < 15%，认为是有效档位

---

### 4. 边界判定

```dart
if (speed < 3 || rpm < 800) {
  return 0;  // 显示空档
}
```

| 参数 | 当前值 | 说明 |
|------|--------|------|
| 最低速度 | 3 km/h | 低于此值显示空档 |
| 最低转速 | 800 rpm | 低于此值显示空档 |

---

### 5. 平滑滤波 (stableFrames)

```dart
if (_stableFrames < 5) {
  return _lastValidGear;  // 未稳定，保持原档位
}
```

- 必须连续 **5 帧** 计算结果一致才切换档位
- 2Hz 采样频率下，约 2.5 秒
- 作用：防止瞬间数据波动导致档位跳动

---

## 校准方法

### 方法1: 通过实际骑行数据反推

在骑行中记录各档位的实际转速和车速：

| 档位 | 转速 (rpm) | 车速 (km/h) |
|------|------------|-------------|
| 1档 | ? | ? |
| 2档 | ? | ? |
| 3档 | ? | ? |
| 4档 | ? | ? |
| 5档 | ? | ? |
| 6档 | ? | ? |

反推公式：
```
实测传动比 = (转速 × 60 × 轮胎周长) / (车速 × 1000)
```

然后计算：
```
实测内部齿轮比 = 实测传动比 ÷ (primaryRatio × finalRatio)
```

### 方法2: 调整误差阈值

如果 4/5/6 档互相跳动，可以：
1. 增大 `_gearThresholds` 值（如从 0.15 改为 0.20）
2. 增加 `stableFrames` 值（如从 5 改为 8）
